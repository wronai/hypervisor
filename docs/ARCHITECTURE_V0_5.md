# Architecture v0.5

Skanowanie usług, graf URI i `nl2uri` są częścią paczek `uri3` / `nl2uri`, a nie rdzenia hypervisora.

```txt
uri3 = discovery, routing, graph, validation
nl2uri = natural language -> URI Tree
hypervisor = policy, registry, lifecycle, deployment
Domain Pack = logika domenowa
Generated Agent = cienki adapter
```
