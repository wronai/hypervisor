# Oferty i wiadomości sprzedażowe

Pakiet wejścia na rynek polski — pierwszych 10 klientów.

## Trzy oferty produktowe

### Oferta 1: Audyt automatyzacji

| | |
|---|---|
| **Cena** | 2 500–5 000 zł netto |
| **Czas** | 1–3 dni |
| **Dla kogo** | Każdy segment — niski próg wejścia |

**Wynik:**

```text
- mapa procesów,
- lista automatyzacji,
- 3 najczęstsze punkty awarii,
- propozycja URI dla procesów,
- mini-demo dashboardu,
- rekomendacja pilota.
```

---

### Oferta 2: Pilot Taskinity dla 3 procesów

| | |
|---|---|
| **Cena** | 8 000–15 000 zł netto (segment zależny — patrz ICP) |
| **Czas** | 7–14 dni |
| **Dla kogo** | Software house, integrator, e-commerce, BPO |

**Wynik:**

```text
- 3 procesy jako URI,
- Web UI + chat,
- API / CLI proof,
- health checks,
- logi ostatnich wykonań,
- incident,
- ticket,
- raport: co padło i ile czasu można oszczędzić.
```

**Oferta wejścia software house:**

> „Podłączymy 3 wasze realne procesy: GitHub Action, webhook/API i skrypt/cron. Pokażemy health, logi, incident i ticket w chacie.”

**Oferta wejścia integrator n8n/Make:**

> „Taskinity jako panel serwisowy dla wdrożeń n8n/Make: status procesów, ostatnie błędy, ticket i rekomendacja naprawy.”

**Oferta wejścia e-commerce:**

> „Monitoring 3 procesów: zamówienia, faktury, zwroty.”

**Oferta wejścia BPO / księgowość:**

> „Mapa procesu faktur + dashboard błędów + ticket dla dokumentów, których automat nie rozumie.”

## Pierwszych 10 klientów — docelowy miks

```text
4  software house’y / małe DevOps / firmy AI automation
2  integratorów n8n / Make / Power Automate
2  e-commerce z własnym magazynem lub ERP
1  biuro rachunkowe / BPO dokumentowe
1  firma produkcyjno-logistyczna (import/eksport danych)
```

**Priorytet kanału:** integrator > software house > końcowy klient e-commerce.

## Wiadomości sprzedażowe (gotowce)

### Do software house’u

```text
Cześć, buduję Taskinity — prosty command center dla automatyzacji, skryptów i agentów AI.

Nie zastępuje n8n, GitHub Actions ani waszych skryptów. Daje jeden widok:
co działa, co padło, gdzie są logi, jaki jest incident i co można naprawić.

Szukam 5 polskich software house’ów do pilotażu.
W 7–14 dni podłączamy 3 realne procesy i pokazujemy je w chacie + dashboardzie.

Czy mogę pokazać 10-min demo na przykładzie:
webhook → błąd API → incident → ticket → propozycja naprawy?
```

### Do integratora n8n / Make

```text
Cześć, Taskinity to warstwa utrzymania dla wdrożeń n8n/Make —
status procesów, ostatnie błędy, ticket i rekomendacja naprawy w jednym panelu.

Szukam 2 integratorów do pilotażu (5–12k zł, 7–14 dni).
Możecie dodać to do oferty SLA dla klientów.

10 min demo — pokażę panel serwisowy na przykładzie webhook + błąd API.
```

### Do e-commerce

```text
Cześć, pomagamy firmom e-commerce ogarnąć awarie automatyzacji:
zamówienia, faktury, ERP, magazyn, kurierzy, marketplace.

Taskinity pokazuje w prostym chacie:
który proces padł, dlaczego, gdzie są logi i co zrobić dalej.

Robimy 14-dniowy pilot na 3 procesach.
Czy warto sprawdzić to na jednym waszym procesie zamówień lub faktur?
```
